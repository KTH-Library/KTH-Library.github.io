What is new?

  • During the autumn, we have primarily worked on a new model with reporting at divisional level, based on currently employed researcher and their publication outputs.

  • Work has been initiated with regards to using Scopus data from Elsevier in order to complement the WoS data.

  • We have upgraded the stack that the software runs on and updated the server environment (with patches, new certificates and API keys).

  • We have separated operational ABM services from other Bibliometrics services and have allowed external users without KTH accounts to log in via the use of auth0

  • We have initiated a migration from Travis - a continuous integration provider Travis - to using GitHub Actions.

New components

A new R package (semanticscholar) allows for experimenting with data integration against this data source which provides open data for publications from many researchers at KTH.
