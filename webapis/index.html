Att deploya ett Plumber-API

Om man vill deploya detta som en container med docker s behver man paketera det. D behvs ytterligare ngra filer:

  • ett startscript

  • en Makefile som anvnd fr att bygga, starta och testa i olika steg (eller man kan gra alla genom att ange make enbart)

  • en Dockerfile - men varfr behvs den? Varfr anvnds inte kthb/kontarion:latest direkt?

  • en docker-compose.yml-fil - som anger en minimal composition fr att kra API:et
